#jQuery Iconize Plugin

## What is it?
It helps to add an overlay image to an image.

## Requirements
jQuery >= 1.4.1

## Compability
ie6+ (supports transparent pngs)

Firefox

Chrome

Safari

Opera

## Usage
	// Simple Usage
	$('.video-link').iconize({
		    iconSrc : 'images/video.png
	});
